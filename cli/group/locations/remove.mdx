<Warning>
  This feature is deprecated. Turso Cloud no longer supports multi-region replication for new users.
</Warning>

You can remove locations from a group using the command below:

```bash
turso group locations remove <group-name> <...location-codes>
```

<Info>

All database replicas belonging to the provided group will be immediately removed from the locations.

</Info>
