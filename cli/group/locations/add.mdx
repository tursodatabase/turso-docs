<Warning>
  This feature is deprecated. Turso Cloud no longer supports multi-region replication for new users.
</Warning>

You can add locations to a group using the following command:

```bash
turso group locations add <group-name> <location-code> [flags]
```

## Flags

| Flag           | Description                                |
| -------------- | ------------------------------------------ |
| `-w`, `--wait` | Wait for group to be ready before exiting. |
