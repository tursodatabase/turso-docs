## Prerequisites

To get the most out of this guide, you'll need to:

- [Install the Turso CLI](/cli/installation)
- [Sign up or login to Turso](/cli/authentication#signup)
- Have an Astro project &mdash; [learn more](https://docs.astro.build/en/install/auto/#1-run-the-setup-wizard)

<Step title="Configure Turso client">

```ts src/turso.ts
import { connect } from "@tursodatabase/serverless";

export const turso = connect({
  url: import.meta.env.TURSO_DATABASE_URL!,
  authToken: import.meta.env.TURSO_AUTH_TOKEN,
});
```

<Note>
  Astro will soon introduce a new ENV API. [Take a
  look](https://docs.astro.build/en/reference/configuration-reference/#experimentalenv).
</Note>

</Step>

<Step title="Execute SQL">

```ts
---
import { turso } from './turso'

const stmt = await turso.prepare('SELECT * FROM table_name')
const rows = await stmt.all()
---
```

</Step>
