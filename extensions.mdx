## Default Extensions

The extensions **enabled by default** in all databases are:

<CardGroup>
  <Card title="JSON" href="https://www.sqlite.org/json1.html">

  </Card>
  <Card title="Full-Text Search" href="https://www.sqlite.org/fts5.html">
    Full-text search and indexing.
  </Card>
  <Card title="R*Tree" href="https://www.sqlite.org/rtree.html">
    Indexing and querying spatial data.
  </Card>
</CardGroup>

## Usage

You can enable all extensions when creating a new database using the [CLI](/cli/db/create) or [API](/api-reference/databases/create):

<CodeGroup>

```bash Turso CLI
turso db create [database-name] --enable-extensions
```

```bash Platform API
curl -L -X POST 'https://api.turso.tech/v1/organizations/{organizationName}/databases' \
  -H 'Authorization: Bearer TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
      "name": "new-database",
      "group": "default",
      "extensions": "all"
  }'
```

</CodeGroup>

If you already have a group created with extensions disabled, you can update all databases in a group to enable extensions:

```bash
turso group update <group-name> --extensions all
```
