## Prerequisites

To get the most out of this guide, you'll need to:

- [Install the Turso CLI and create a database].
- [Have an Astro project].

<Steps>

<Step title="Install the libSQL JavaScript SDK">

<CodeGroup>

```bash npm
npm install @libsql/client
```

```bash pnpm
pnpm add @libsql/client
```

```bash yarn
yarn add @libsql/client
```

</CodeGroup>

</Step>

<Step title="Set up the database credentials.">

Get the database URL.

```bash
turso db show --url my-database
```
Get the database authentication token.

```bash
turso db tokens create my-database
```

Assign acquired credentials to the environment variables inside `.env.local`.

```bash
TURSO_DB_URL=<DB-URL>
TURSO_DB_AUTH_TOKEN=<DB-AUTH-TOKEN>
```
</Step>

<Step title="Initiate a LibSQL Client.">

```javascript src/turso.ts

import { createClient } from '@libsql/client/web';

export const turso = tursoClient() {
  const url = import.meta.env.VITE_TURSO_DB_URL?.trim();
  if (url === undefined) {
    throw new Error('TURSO_DB_URL is not defined');
  }

  const authToken = import.meta.env.VITE_TURSO_DB_AUTH_TOKEN?.trim();
  if (authToken === undefined) {
    throw new Error('TURSO_DB_AUTH_TOKEN is not defined');
  }

  return createClient({
    url,
    authToken,
  });
}
```
</Step>

<Step title="Fetch data from Turso.">

```javascript
---
import { turso } from '../../turso'

const { rows } = await tursoClient.execute('SELECT * FROM posts')
---

<ul>
  {rows.map((post) => (
    <li>{post.title}</li>
  ))}
</ul>
```
</Step>

</Steps>
