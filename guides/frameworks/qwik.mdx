## Prerequisites

To get the most out of this guide, you'll need to:

- [Install the Turso CLI and create a database].
- [Have a Qwik app].

<Steps>

<Step title="Add the Turso integration">

<CodeGroup>

```bash npm
npm run qwik add turso
```

```bash pnpm
pnpm qwik add turso
```

```bash yarn
yarn qwik add turso
```

</CodeGroup>

</Step>

<Step title="Set up the database credentials.">

Get the database URL.

```bash
turso db show --url my-database
```
Get the database authentication token.

```bash
turso db tokens create my-database
```

Assign acquired credentials to the environment variables inside `.env.local`.

```bash
PRIVATE_TURSO_DB_URL=<DB-URL>
PRIVATE_TURSO_DB_AUTH_TOKEN=<DB-AUTH-TOKEN>
```
</Step>

<Step title="Fetch data from Turso.">

```javascript
import { tursoClient } from '~/utils/turso';

export const useFrameworks = routeLoader$(async (requestEvent: RequestEventBase) => {
  const response = await tursoClient(requestEvent["env"]).execute("select * from my-table");

  return {
    items: response.rows,
  }
})
```
</Step>

</Steps>
