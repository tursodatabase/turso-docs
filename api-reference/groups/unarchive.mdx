sidebarTitle: Unarchive

---

<Info>

Databases get archived after 10 days of inactivity for users on a free plan &mdash; [learn more](/features/scale-to-zero). You can unarchive inactive groups using the API.

</Info>

<RequestExample>

```bash cURL
curl -L -X POST 'https://api.turso.tech/v1/organizations/{organizationName}/groups/{groupName}/unarchive' \
-H 'Authorization: Bearer TOKEN' \
```

</RequestExample>
