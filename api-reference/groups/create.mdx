<Warning>Creating more than one group is limited to paid plans.</Warning>

<RequestExample>

```bash cURL
curl -L -X POST 'https://api.turso.tech/v1/organizations/{organizationName}/groups' \
-H 'Authorization: Bearer TOKEN' \
-H 'Content-Type: application/json' \
-d '{
    "name": "new-group",
    "location": "lhr"
}'
```

```ts Node.js
import { createClient } from "@tursodatabase/api";

const turso = createClient({
  org: "...",
  token: "",
});

const group = await turso.groups.create("new-group", {
  location: "lhr",
  extensions: ["vector", "uuid"], // 'all'
});
```

</RequestExample>
