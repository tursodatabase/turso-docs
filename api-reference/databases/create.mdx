<RequestExample>

```bash cURL
curl -L -X POST 'https://api.turso.tech/v1/organization/{name}/databases' \
  -H 'Authorization: Bearer TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
      "name": "new-database",
      "group": "default"
  }'
```

```ts Node.js
import { createClient } from "@turso/api";

const turso = createClient({
  organization: "...",
  token: "",
});

const database = await turso.databases.create("new-database", "default");
```

</RequestExample>
