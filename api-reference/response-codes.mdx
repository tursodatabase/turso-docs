title: Errors
description:
---

The Turso API will respond with an HTTP status code, and in the event of an error, return an error message in the response body.

| Code  | Description                                                            |
| ----- | ---------------------------------------------------------------------- |
| `200` | `OK` &mdash; Successful request                                        |
| `401` | `Unauthorized` &mdash; Invalid or expired auth token                   |
| `402` | `Payment required` &mdash; Check you have an active subscription       |
| `403` | `Forbidden` &mdash; You do not have permission to access this resource |
| `409` | `Conflict` &mdash; Resource already exists                             |
